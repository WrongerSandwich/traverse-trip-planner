# Archived design specs

Specs in this folder describe work that has already shipped (or has been
explicitly superseded by a later spec). Their implementation may have
diverged from the original design — **treat the code as the source of
truth** and use these documents as historical context for the why behind
the current implementation.

The `docs/superpowers/specs/` folder above is reserved for **active**
design work — proposals not yet merged, or specs whose implementation
is still in progress.

See `git log` on each file for the PR(s) that landed the work.
